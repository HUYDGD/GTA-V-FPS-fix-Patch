# GTA-V-FPS-fix-Patch
<h3>GTA V settings.xml patch</h3>
A Patch to GTA V settings.xml file.

1.  .NET Framework 4.5 or above.
2.  Visual Studio 2015 update 3 or above.


1.  Allows To run GTA V on low end PCs.
